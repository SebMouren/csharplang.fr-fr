### <a name="accessibility-domains"></a><span data-ttu-id="37cab-271">Domaines d’accessibilité</span><span class="sxs-lookup"><span data-stu-id="37cab-271">Accessibility domains</span></span>

<span data-ttu-id="37cab-281">Le domaine d’accessibilité d’un membre imbriqué `M` déclaré dans un type `T` au sein d’un programme `P` est défini comme suit (en notant que `M` peut éventuellement être lui-même un type) :</span><span class="sxs-lookup"><span data-stu-id="37cab-281">The accessibility domain of a nested member `M` declared in a type `T` within a program `P` is defined as follows (noting that `M` itself may possibly be a type):</span></span>
